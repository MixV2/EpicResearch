# RefundMtxPurchase
Refunds something that was bought by mtx.

## Attributes
Compatible Profiles: `common_core`  
Supported MCPs: `fortnite`

## Payload
```json
{
    "purchaseId": "",
    "quickReturn": true
}
```

## Parameters
- `purchaseId`: purchase ID provided from mcp
- `quickReturn`: quickly refund a item that was recently bought
